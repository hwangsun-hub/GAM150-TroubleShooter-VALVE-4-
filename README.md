Compilation Instructions
Install Visual Studio with the Desktop development with C++ workload.
Download and extract the project ZIP file.
Open the project folder.
Double-click the `.sln` file to open the project in Visual Studio.
Build the project by pressing Ctrl + Shift + B.
Run the game by pressing Ctrl + F5.
If an error occurs, go to [Project Properties] > [Configuration Properties] > [General] > [Platform Toolset] in Visual Studio and change it to the currently installed version (for example, v143 for Visual Studio 2022).

- Explanation of what the game is and how to play
This game is a platformer where the player wins by reaching the flag.
If you clear 18 maps, the game is finished.

You can move using the left and right arrow keys, and jump with the space bar.
You can shoot trouble with "Z"key.
You can restart the stage by pressing the R key.

The F1 key is a debugging key that shows the hitboxes of objects.
You can skip the map by pressing the F10 key.
You can go to mainmenu by pressing 1 key.
